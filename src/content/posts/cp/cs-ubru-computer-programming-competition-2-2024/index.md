‡∏Ç‡πâ‡∏≠‡∏ô‡∏µ‡πâ‡∏à‡∏∞‡πÄ‡∏õ‡πá‡∏ô‡πÄ‡∏£‡∏∑‡πà‡∏≠‡∏á set ‡∏Ñ‡∏£‡∏±‡∏ö ‡πÇ‡∏î‡∏¢‡∏°‡∏±‡∏ô‡πÄ‡∏£‡∏µ‡∏¢‡∏Å‡∏ß‡πà‡∏≤ symmetric difference ‡∏ã‡∏∂‡πà‡∏á python ‡∏°‡∏±‡∏ô‡∏ó‡∏≥‡∏á‡πà‡∏≤‡∏¢‡∏°‡∏≤‡πÅ‡∏Ñ‡πà `a ^ b` ‡πÅ‡∏ï‡πà‡πÉ‡∏ô‡∏ï‡∏≠‡∏ô‡πÅ‡∏Ç‡πà‡∏á‡∏ú‡∏°‡∏ô‡∏∂‡∏Å‡πÑ‡∏°‡πà‡∏≠‡∏≠‡∏Å‡πÄ‡∏•‡∏Ç‡πÄ‡∏Ç‡∏µ‡∏¢‡∏ô‡∏™‡∏π‡∏ï‡∏£‡πÄ‡∏≠‡∏≤‡∏Ñ‡∏£‡∏±‡∏ö

```py
a = set(map(int, input().split(" ", 1)[-1].replace(" ", "").split(",")))
b = set(map(int, input().split(" ", 1)[-1].replace(" ", "").split(",")))
l = sorted(list(a.union(b) - a.intersection(b)))
if len(l) == 0:
    print("No Differences")
else:
    print(f"{len(l)} {', '.join(map(str, l))}")
```

‡πÉ‡∏ô‡∏ï‡∏≠‡∏ô‡πÅ‡∏£‡∏Å run ‡πÉ‡∏ô local ‡∏ñ‡∏π‡∏Å‡∏Ñ‡∏£‡∏±‡∏ö‡πÅ‡∏ï‡πà submit ‡πÑ‡∏õ‡πÅ‡∏•‡πâ‡∏ß‡∏ú‡∏¥‡∏î ‡∏ú‡∏°‡πÄ‡∏•‡∏¢‡πÑ‡∏°‡πà‡∏°‡∏±‡πà‡∏ô‡πÉ‡∏à‡∏ß‡πà‡∏≤ input ‡∏°‡∏µ‡∏≠‡∏∞‡πÑ‡∏£‡∏≠‡∏¢‡πà‡∏≤‡∏á‡∏≠‡∏∑‡πà‡∏ô‡πÄ‡∏ä‡πà‡∏ô‡∏ï‡∏±‡∏ß‡πÄ‡∏•‡∏Ç‡πÑ‡∏´‡∏° ‡∏´‡∏£‡∏∑‡∏≠‡∏¢‡∏±‡∏á‡πÑ‡∏á‡∏ú‡∏°‡πÄ‡∏•‡∏¢‡∏•‡∏≠‡∏á‡πÅ‡∏õ‡∏∞ `str.isalpha` ‡∏î‡∏π ‡∏™‡∏£‡∏∏‡∏õ‡πÑ‡∏î‡πâ‡∏ú‡∏• ‡πÅ‡∏•‡πâ‡∏ß‡∏°‡∏≤‡∏£‡∏π‡πâ‡∏ó‡∏µ‡πà‡∏´‡∏•‡∏±‡∏á‡∏ß‡πà‡∏≤ testcase ‡πÄ‡∏õ‡πá‡∏ô‡πÑ‡∏ü‡∏•‡πå CRLF (‡∏°‡∏±‡∏ô‡∏°‡∏µ `\r` ‡∏ã‡∏∂‡πà‡∏á‡∏°‡∏±‡∏ô‡∏´‡∏•‡∏∏‡∏î‡πÄ‡∏Ç‡πâ‡∏≤‡πÑ‡∏õ‡πÉ‡∏ô Counter)

```py
from collections import Counter

while True:
    s = input()
    if s == "stop":
        break

    if len(s) < 2:
            print("NOT pair isogram")
            continue

    c = Counter([x for x in s.lower() if x.isalpha()])
    for _, v in c.items():
        if v != 2:
            print("NOT pair isogram")
            break
    else:
        print("pair isogram")
```

‡∏à‡∏∞‡πÄ‡∏´‡πá‡∏ô‡∏ß‡πà‡∏≤‡πÄ‡∏°‡∏∑‡πà‡∏≠‡πÅ‡∏ó‡∏ô A ‡πÄ‡∏õ‡πá‡∏ô 1 ‡πÅ‡∏•‡πâ‡∏ß‡∏•‡∏∞‡∏Å‡πá ‡πÄ‡∏°‡∏∑‡πà‡∏≠ Z = 26 ‡∏ñ‡πâ‡∏≤‡πÄ‡∏õ‡πá‡∏ô 27 ‡∏à‡∏∞‡∏Å‡∏•‡∏≤‡∏¢‡πÄ‡∏õ‡πá‡∏ô AA ‡πÅ‡∏ï‡πà‡πÄ‡∏î‡∏µ‡πã‡∏¢‡∏ß‡∏Å‡πà‡∏≠‡∏ô A ‡πÄ‡∏õ‡πá‡∏ô 1 ‡∏ô‡∏µ‡πâ‡∏Ñ‡∏∑‡∏≠‡∏™‡∏≤‡πÄ‡∏´‡∏ï‡∏∏‡∏ß‡πà‡∏≤‡∏ó‡∏≥‡πÑ‡∏°‡∏Å‡∏≤‡∏£‡πÅ‡∏õ‡∏•‡∏á‡πÄ‡∏•‡∏Ç‡∏ê‡∏≤‡∏ô‡∏õ‡∏Å‡∏ï‡∏¥‡πÉ‡∏ä‡πâ‡πÑ‡∏°‡πà‡πÑ‡∏î‡πâ ‡πÄ‡∏£‡∏≤‡∏à‡∏∂‡∏á‡∏ï‡πâ‡∏≠‡∏á‡∏´‡∏≤‡∏ß‡∏¥‡∏ò‡∏µ‡πÅ‡∏õ‡∏•‡∏á‡∏≠‡∏∑‡πà‡∏ô‡πÜ‡∏ó‡∏µ‡πà‡πÑ‡∏°‡πà‡πÉ‡∏ä‡πà‡∏Å‡∏≤‡∏£‡∏Ñ‡∏≥‡∏ô‡∏ß‡∏ô

‡∏ú‡∏°‡∏à‡∏∂‡∏á‡πÄ‡∏•‡∏∑‡∏≠‡∏Å‡∏ó‡∏µ‡πà‡∏à‡∏∞ loop ‡∏´‡∏¢‡∏¥‡∏ö‡πÄ‡∏•‡∏Ç‡∏≠‡∏≠‡∏Å‡∏°‡∏≤‡∏Å‡πâ‡∏≠‡∏ô‡πÉ‡∏´‡∏ç‡πà‡πÉ‡∏ô‡πÅ‡∏ï‡πà‡∏•‡∏∞‡∏´‡∏•‡∏±‡∏Å‡∏à‡∏ô‡∏°‡∏±‡∏ô‡∏´‡∏°‡∏î

```py
while True:
    row, col = map(int, input().replace("r", "").split("c"))
    if row == 0 and col == 0:
        break

    x = col - 1
    d = ""
    A = ord("A")
    i = 1
    while True:
        c = x % (26 ** i)
        x = (x - c) - 1 # Adjust to 0-based index
        d = chr((c // (26 ** (i - 1))) + A) + d
        if x <= 0:
            break
        i += 1

    print(f"{d}{row}")
```

**Encryption Formula :** $$C_i = E_K(M_i) = (M_i + K_{i \bmod m}) \mod \ell$$

**Decryption Formula :** $$M_i = D_K(C_i) = (C_i - K_{i \bmod m}) \mod \ell$$

$$M_i$$: Offset of the i-th character of the plaintext M in the alphabet Œ£. \
$$C_i$$: Offset of the i-th character of the ciphertext C. \
$$K_{i \bmod m}$$: Offset of the corresponding key character (cyclically repeated). \
$$‚Ñì$$: Length of the alphabet (e.g., ‚Ñì = 26 for English). \
$$m$$: Length of the key.

```py
t = input()
a = ord("a")
A = ord("A")
k = [ord(x.lower()) - a for x in input()]

c = ""
ki = 0
kl = len(k)
for i in t:
    if i.isalpha():
        if i.islower():
            c += chr((((ord(i) - a) + k[ki]) % 26) + a)
        else:
            c += chr((((ord(i) - A) + k[ki]) % 26) + A)
        ki = (ki + 1) % kl
    else:
        c += i

print(c)
```
